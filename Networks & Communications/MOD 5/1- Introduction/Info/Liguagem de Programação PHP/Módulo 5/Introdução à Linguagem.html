Introdução à Linguagem

PHP deriva das palavras Personal Home Page.

PHP é uma Linguagem de Script do Lado do Servidor, de código aberto, muito popular e amplamente usada para escrever páginas dinâmicas para a Web.

PHP foi originalmente criado por Rasmus Lerdorf em 1994.

O código apresentado neste tutorial foi testado e validado com a versão PHP 8.

PHP é uma linguagem muito poderosa, fácil de aprender e utilizar.

O que se pode fazer com o PHP?

  • Gerar páginas e ficheiros dinamicamente.
  • Criar, abrir, ler, gravar e fechar ficheiros no servidor.
  • Recolher dados de um formulário da web, como informações dos utilizadores, e-mail, número de telefone, etc.
  • Enviar e-mails para os utilizadoress do teu site.
  • Enviar e receber cookies para analisar qual o visitante do teu site.
  • Armazenar, apagar e modificar informações numa base de dados.
  • Restringir o acesso não autorizado ao teu site.
  • Criptografar os dados para transmissão segura pela Internet.

A lista não termina aqui, existem muitas outras coisas interessantes que pode fazer com o PHP. Podes melhorar os teus conhecimentos explorando as diferentes secções deste tutorial.
